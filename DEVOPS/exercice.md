
Deployer 5 conteneurs à partir de playbook avec des conditions localement, une tache seulement, une seule commande

```
pip install docker
```

Sans cela Ansible ne sais pas parler avec docker

on crée une liste (un tableau) qui contient les caractéristiques de nos 5 conteneurs. C'est l'étape de **conception**.

- On donne un **nom** unique.
    
- On choisit une **image** (OS ou service) différente pour chacun.
